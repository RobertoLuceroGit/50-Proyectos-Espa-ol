# 50 proyectos en español


### Descripción

**Saludos a todos.**
Este repositorio en español, son un compilado de códigos en:
- HTML
- CSS
- JavaScript

Las mismas están basadas en el siguiente repositorio:

`<Link de repositorio>` https://github.com/bradtraversy/50projects50days

Esta, se basa en un curso de 50 proyectos en 50 días.

`<Link del curso>` https://www.udemy.com/course/50-projects-50-days

`<Link info del curso>` https://50projects50days.com

--------------------------
### ¿Por que nace este repositorio?

Este repositorio nace con la necesidad de práctica y entendimiento de los códigos.

Para ello, se buscó una traducción de variables al idioma español, y una adición de comentarios para entendimiento del funcionamiento de las líneas codificadas. 

Aun así, se han añadido detalles personales para no ser una copia descarada del repositorio original.

--------------------------
